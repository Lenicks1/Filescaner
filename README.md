«Сканер файлов»
Приложение с открытым исходным кодом.
Сканер файлов — это простое приложение для просмотра и открытия файлов в выбранной папке.
Сканирование: 
Указываете путь к папке (или выбираете через «Обзор…»), нажимаете «Сканировать» — все файлы папки и подпапок выводятся списком.
Список файлов:
Тёмная тема, цветное отображение диска, папок и имён файлов, справа показан размер файла.
Сортировка и поиск: 
Можно сортировать файлы по имени, дате, размеру и типу, а также фильтровать их по части имени.
Запуск файлов:
Двойной клик по строке или кнопка «Открыть» запускает файл в стандартной программе.
История: 
Есть вкладка с историей поисков (сохраняется между запусками) и вкладка истории запусков последних открытых файлов.
